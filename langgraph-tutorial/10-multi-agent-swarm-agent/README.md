# Multi-Agent SWARM

## 개요

OpenAI의 SWARM 패턴을 활용한 멀티 에이전트 시스템을 구현하는 프로젝트입니다. 경량화되고 유연한 에이전트 간 핸드오프(handoff) 메커니즘을 통해 동적인 협업을 구현합니다.

## 주요 내용

- SWARM 패턴의 개념과 특징
- 에이전트 간 핸드오프 메커니즘
- 동적 에이전트 라우팅
- 컨텍스트 유지 및 전달
- 경량 멀티 에이전트 오케스트레이션

- SWARM 패턴의 핵심 개념과 장점 이해

- 경량 멀티 에이전트 시스템 설계 방법 습득
