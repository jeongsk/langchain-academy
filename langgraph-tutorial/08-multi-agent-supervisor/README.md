# Multi-Agent Supervisor

## 개요

감독자(Supervisor) 패턴을 활용한 멀티 에이전트 시스템을 구현하는 프로젝트입니다. 중앙 감독자가 여러 작업자 에이전트를 조율하고 작업을 분배하는 계층적 에이전트 아키텍처를 구축합니다.

## 주요 내용

- Supervisor 패턴 아키텍처
- 중앙 집중식 작업 분배
- 작업자 에이전트 관리 및 모니터링
- 동적 작업 할당 및 우선순위 관리
- 에이전트 간 조정 및 충돌 해결

- 효율적인 작업 분배 및 조율 메커니즘 구현
